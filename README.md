# BigLinux Colors

[BigLinux](https://www.biglinux.com.br) color scheme for:

- [KDE](/kde/README.md)
- [Konsole](/konsole/README.md)

## Install

```sh
chmod +x setup.sh && ./setup.sh
```

## Location

```sh
# Colors
~/.local/usr/share/color-schemes/

# Konsole
~/.local/usr/share/konsole/
```
